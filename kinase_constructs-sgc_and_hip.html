Kinase expression constructs

Information

Custom selection procedure:

  • All kinase domain targets which have matching plasmids with < 40 extraneous residues
  • Plus all kinase domain targets which have a matching PDB construct with an authenticity score of > 0
  • While filtering out kinase domain targets with DB_target_rank > 450 (this removes the lowest-ranked of the 498 kinase domain targets, many of which are pseudokinases, or have unusually long/short kinase domains)

And again in pseudocode notation:

( ( plasmid_nextraneous_residues < 40 ) or ( nPDBs > 0 and top_pdb_auth_score > 0 ) ) and DB_target_rank < 450

Description of table fields with details of selection procedure:

targetID Kinase domain targets are given an identifier consisting of UniProt entry name followed by the kinase domain index (since some kinases have two catalytic domains)
DB_target_rank Target rank (based on metrics such as number of bioassays, number of PDBs, number of publications, % mutation rate in sequenced tumor samples)
plasmid_source Two plasmid libraries are currently available to the Chodera lab - SGC Oxford and HIP pJP1520
plasmid_ID For a given target, all matching plasmids are aligned and sorted based on nconflicts and nextraneous_residues (see below), and the top-ranked plasmid is selected (click the targetID to see the alignment)
plasmid_nconflicts Number of conflicts between plasmid and UniProt sequence in the region of the target domain
plasmid_nextraneous_residues Number of plasmid residues outside the target domain region
nPDBs Number of PDB constructs with an E coli expression system annotation
top_pdb_ID Matching PDB constructs are ranked by auth_score (see below), nconflicts, and nextraneous_residues, and the top-ranked PDB (and chain ID) is selected
top_pdb_expr_tag Type of expression tag detected for top-ranked PDB construct
top_pdb_auth_score Authenticity score assigned to top-ranked PDB construct based on the expression tag detected. Currently this is only used to distinguish between constructs with expression tags (auth_score > 0) and those without (auth_score = 0)
top_pdb_nextraneous_residues Number of PDB construct residues outside the target domain region
family UniProt protein family annotation
